# 功能需求文档

## MVP 功能（第一版必须完成）

### 1. 备考计划生成器
**优先级**：P0（最高）

**需求描述**：
- 用户输入考试日期，系统自动计算距离天数
- 根据备考时长（<30天、30-60天、60-90天、>90天）生成不同学习计划
- 每日学习时长可自定义（30分钟 - 5小时）
- 自动划分阶段：基础期、强化期、冲刺期
- 生成今日任务清单（可勾选完成）

**验收标准**：
- [x] 可以选择考试日期
- [x] 自动显示距离考试天数和当前阶段
- [x] 今日任务清单可勾选
- [x] 数据保存到 localStorage

---

### 2. 雅思词汇记忆系统
**优先级**：P0（最高）

**需求描述**：
- 内置雅思核心词汇（MVP 先使用示例词表，后续扩展到高频词库）
- 使用 SM-2 间隔重复算法（类 Anki）
- 卡片式学习：正面单词、背面释义+例句
- 四个按钮：不认识、模糊、认识、熟练
- 每日新词数量可自定义（10-50词）
- 显示学习进度：已学、复习中、已掌握
- 不认识/模糊的词自动进入薄弱词队列，优先反复练习

**验收标准**：
- [x] 可以查看词汇卡片（翻转效果）
- [x] 点击按钮后根据 SM-2 算法安排下次复习时间
- [x] 每日新词数量可调整
- [x] 学习进度可视化
- [x] 扩展 IELTS-4000 完整词库来源
- [x] 不认识/模糊自动进入薄弱词队列
- [ ] 为导入词条逐步补充中文释义、音标和例句

**数据结构**：
```javascript
{
  vocab: {
    "word-id": {
      word: "academic",
      meaning: "学术的；理论的",
      example: "He has an academic background.",
      theme: "education",
      easeFactor: 2.5,      // SM-2 难度因子
      interval: 0,          // 复习间隔（天）
      nextReview: "2026-07-03",
      reviewCount: 0,
      lastReview: null
    }
  }
}
```

## 第二阶段功能

### 3. 四科训练记录
**优先级**：P1

- 听力、阅读、写作、口语训练记录
- 记录日期、时长、材料来源、成果
- 添加复盘笔记
- 每周统计训练时长

### 4. 错题本
**优先级**：P1

- 记录错题（科目、题型、错因）
- 标记是否已解决
- 错题复习提醒

### 5. 模考记录
**优先级**：P1

- 记录模考日期、套题来源
- 输入 L/R/W/S 分数，自动计算 Overall
- 模考分数趋势图

---

## 第三阶段功能

### 6. 阅读刷题系统
**优先级**：P2

- 内置阅读 Passage（至少 5 篇）
- 支持题型：判断题、匹配题、选择题、填空题
- 计时器
- 自动判分
- 查看解析
- 错题自动入错题本

**当前进度**：
- [x] 建立阅读题库数据结构
- [x] 加入 1 篇项目原创 Passage 示例
- [ ] 实现答题 UI、计时、判分、错题入库

### 7. 听力刷题系统
**优先级**：P2

- 音频播放器
- 内置听力 Section（至少 3 套）
- 支持题型：填空、选择、地图
- 查看原文
- 精听功能

**当前进度**：
- [x] 建立听力题库数据结构
- [x] 加入 1 套项目原创 transcript 示例
- [ ] 实现音频播放器、答题 UI、判分、错题入库
